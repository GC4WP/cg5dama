## 50+1
> 个人+协同作业

按照以往的经验, 最好玩儿的协同就是:

- 0: 大家都先各自写出第一个屎稿, 发现到 Issue 中
- 1: 快速用 emoji 点评, 得赞心最多者, 作为团队底稿
- 2: 原作, 作为合稿人, 将文稿发布到仓库中, 其它成员, 通过 Commit-Comments 来点评
- 3: 根据意见,快速和合至少3个版本, 即可获得团队作业
得以发布.
